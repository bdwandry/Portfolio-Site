# Getting project started
### 1. From Terminal: `$ git clone https://github.com/bdwandry/Portfolio-Site.git`
### 2. From Terminal: `$ npm install`
### 3. From Terminal: `$ npm start`
### 4. Navigate to [http://localhost:3000](http://localhost:3000)
